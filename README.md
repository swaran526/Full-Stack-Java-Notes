# Full-Stack Java Notes

Practice code and notes from learning full-stack Java web development — covering basic Java web apps, frontend pages, and JavaScript exercises.

## 📁 Repository Structure

```
Full-Stack-Java-Notes/
├── FirstWebApp/    # First Java web application (introductory project)
├── fronted/         # Frontend practice pages (HTML/CSS)
└── javascript/       # JavaScript practice and exercises
```

## 🛠️ Tech Stack

- **Backend:** Java
- **Frontend:** HTML, CSS, JavaScript

## 📋 What's Inside

- **FirstWebApp/** — An introductory Java web application used to practice the basics of full-stack Java development.
- **fronted/** — Standalone frontend pages built while practicing HTML/CSS layout and styling.
- **javascript/** — Small JavaScript exercises and practice snippets.

## 🚀 Getting Started

```bash
git clone https://github.com/swaran526/Full-Stack-Java-Notes.git
cd Full-Stack-Java-Notes
```

Then open the relevant folder:
- For **FirstWebApp**, open it in your Java IDE (Eclipse/IntelliJ) or deploy it to a servlet container (e.g., Apache Tomcat) if it's a JSP/Servlet-based app.
- For **fronted** and **javascript**, simply open the HTML files directly in a browser.

## 📄 About

This repository serves as a personal notes and practice space while learning full-stack Java web development.
